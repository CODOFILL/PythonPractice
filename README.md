# PythonPractice
Python Practice for algos

## Run tasks

Running tasks template:
```
inv {TASK_NAME} --target={TARGET_NAME}
```

Where,
{TASK_NAME}:
 - bench: (./benchmarks/*.py)
 - example: (./examples/*.py)

{TARGET_NAME}:
Py module name, for ex.:
```
inv bench --target=bitslicing
```
> Execute run() from ./benchmarks/bitslicing.py
